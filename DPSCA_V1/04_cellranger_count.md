# Non-Model Species Single-Cell RNA-seq CellRanger Analysis Pipeline

## Overview

This pipeline describes how to use CellRanger for single-cell RNA-seq reference genome construction, quantitative analysis, and quality control for non-model plant species. It is applicable to transcript reference sequences generated by TransDecoder.

## Prerequisites

### Software Requirements
- **CellRanger**: v10.0.0+ (10x Genomics official software)
- **STAR**: Built-in with CellRanger
- **Seurat** (R package): For downstream dimensionality reduction and clustering
- **Scanpy** (Python package): For quality control

### Input Files
1. **Reference sequence**: `reference.fasta` - Transcript sequences (FASTA format)
2. **Gene annotation**: `reference.gtf` - GTF format annotation file
3. **Sequencing data**: 10x Genomics format FASTQ files

## Pipeline Steps

### Step 1: Prepare Reference Files

Prepare CellRanger compatible reference files from the TransDecoder output:

```bash
# Input files (from TransDecoder)
TRANSDIR="/path/to/transdecoder_output"
CELLRANGER_INPUT="/path/to/cellranger_input"

# Required files:
# - filtered_orfs.gff3 (ORF prediction results)
# - genes.fasta (Transcript sequences)
```

GTF file format requirements:
- Must contain `transcript_id` and `gene_id` attributes
- Recommended format: `transcript_id "xxx"; gene_id "xxx";`

### Step 2: Build Reference Genome (mkref)

```bash
# Set paths
CELLRANGER="/path/to/cellranger-10.0.0/cellranger"
WORKDIR="/path/to/workdir"
FASTA="/path/to/reference.fasta"
GTF="/path/to/reference.gtf"

# Run mkref
cd ${WORKDIR}/reference
${CELLRANGER} mkref \
    --genome=Species_ref \
    --fasta=${FASTA} \
    --genes=${GTF}
```

**Estimated time**: 5-10 minutes (depending on the number of transcripts)

### Step 3: Quantitative Analysis (count)

```bash
# FASTQ file path
FASTQS="/path/to/fastq_directory"
SAMPLE="SampleName"  # FASTQ file prefix

# Run count
cd ${WORKDIR}/count
${CELLRANGER} count \
    --id=${SAMPLE} \
    --create-bam=true \
    --transcriptome=${WORKDIR}/reference/Species_ref \
    --fastqs=${FASTQS} \
    --sample=${SAMPLE} \
    --localcores=40 \
    --localmem=800
```

**Parameter Description**:
- `--id`: Output directory name
- `--create-bam`: Generate BAM file (required)
- `--transcriptome`: Path to the reference genome generated by mkref
- `--fastqs`: Directory where FASTQ files are located
- `--sample`: Sample name (corresponds to FASTQ file prefix)
- `--localcores`: Number of CPU cores used
- `--localmem`: Memory used (GB)

**FASTQ file naming convention**:
```text
{SampleName}_S1_L001_R1_001.fastq.gz
{SampleName}_S1_L001_R2_001.fastq.gz
```

**Estimated time**: 2-4 hours (depending on data volume, ~100GB of data takes about 2.5 hours)

### Step 5: Quality Control (Scanpy/Seurat)

#### Using Scanpy for QC

```python
import scanpy as sc

# Load data
adata = sc.read_10x_mtx(
    'count/{SampleName}/outs/filtered_feature_bc_matrix/',
    var_names='gene_symbols',
    cache=True
)

# Calculate QC metrics
adata.var['mt'] = adata.var_names.str.startswith('MT-')  # Mitochondrial genes
sc.pp.calculate_qc_metrics(adata, qc_vars=['mt'], inplace=True)

# MAD-based adaptive threshold filtering
def mad_filter(adata, metric, n_mads=5):
    median = np.median(adata.obs[metric])
    mad = np.median(np.abs(adata.obs[metric] - median))
    threshold = median + n_mads * mad
    return adata.obs[metric] < threshold

# Filter low-quality cells
adata = adata[filter_mask].copy()

# Normalization
sc.pp.normalize_total(adata, target_sum=1e4)
sc.pp.log1p(adata)

# HVG selection
sc.pp.highly_variable_genes(adata, n_top_genes=2000)
```

#### Using Seurat for Dimensionality Reduction and Clustering

```r
library(Seurat)

# Load data
seurat_obj <- Read10X(data.dir = "count/{SampleName}/outs/filtered_feature_bc_matrix/")
seurat_obj <- CreateSeuratObject(counts = seurat_obj, project = "Species")

# Normalization
seurat_obj <- NormalizeData(seurat_obj)
seurat_obj <- FindVariableFeatures(seurat_obj, nfeatures = 2000)

# PCA dimensionality reduction
seurat_obj <- ScaleData(seurat_obj)
seurat_obj <- RunPCA(seurat_obj, npcs = 50)

# UMAP dimensionality reduction
seurat_obj <- RunUMAP(seurat_obj, dims = 1:30)

# Clustering
seurat_obj <- FindNeighbors(seurat_obj, dims = 1:30)
seurat_obj <- FindClusters(seurat_obj, resolution = 0.5)

# Visualization
DimPlot(seurat_obj, reduction = "umap", group.by = "seurat_clusters")
```

## Key QC Metrics

| Metric | Normal Range | Description |
|------|----------|------|
| Estimated Number of Cells | >1000 | Typical value for 10x data |
| Mean Reads per Cell | >50,000 | Deep sequencing |
| Median Genes per Cell | >500 | Gene detection rate |
| Reads Mapped to Transcriptome | >40% | Lower when using a transcript reference |
| Valid Barcodes | >90% | Sequencing quality metric |
| Sequencing Saturation | >50% | Library complexity |

## Troubleshooting

### Issue 1: mkref fails - GTF format error

**Symptom**: `GTF file does not contain required attributes`

**Solution**:
```bash
# Check GTF format
head reference.gtf
# Ensure it contains transcript_id and gene_id attributes

# Convert from TransDecoder GFF3
python prepare_cellranger_reference.py \
    --gff3 filtered_orfs.gff3 \
    --fasta genes.fasta \
    --output cellranger_input/
```

### Issue 2: Low count mapping rate

**Symptom**: Mapped to transcriptome < 30%

**Possible Causes**:
1. Poor reference sequence quality
2. Sample contamination
3. Sequencing quality issues

**Solutions**:
1. Check reference sequence integrity
2. Use FastQC to evaluate sequencing quality
3. Consider using a full-length transcriptome as the reference

### Issue 3: Insufficient memory

**Symptom**: `Out of memory` error

**Solution**:
```bash
# Increase memory allocation
cellranger count ... --localmem=1000  # 1TB

# Or reduce parallel cores
cellranger count ... --localcores=20 --localmem=500
```

## Special Considerations for Non-Model Species

1. **Reference Genome**: Transcript sequences are usually used instead of whole genomes
2. **Mapping Rate**: Expected mapping rate is relatively low (40-60%)
3. **Functional Annotation**: Subsequent homologous gene annotation is required
4. **Cell Types**: Lack of standard marker genes requires experimental verification
